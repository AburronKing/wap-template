# 移动端页面开发模板

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### 目录层级
```text
├── public
├── src
│     ├── api           api接口
│     ├── assets        静态资源目录
│     ├── components    组件
│     ├── router        路由配置
│     ├── store         Vuex配置
│     ├── utils         工具函数目录
│     └── views         页面
```
